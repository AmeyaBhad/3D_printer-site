# 3D_printer-site
an ecommerce website for selling 3d printed objects. Users can browse products, place orders, and manage accounts, while the backend handles authentication, product management, and order processing.

---

#Tech Stack Used:

## Frontend
- **React **
- **Tailwind CSS**
- **React Router DOM**
- **Framer Motion**

## Backend
- **Java**
- **Spring Boot 3.2.2**
- **Spring Security**
- **Maven**
- **REST APIs**

---

# Features
- Browse 3D printed products
- View product details
- Place orders
- Product management APIs
- Order management APIs
- Authentication & authorization using Spring Security
- Data seeding for initial setup
